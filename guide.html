📊 文件轉換方法

Markdown → PDF

方法 1：使用線上工具 - 網站：https://md2pdf.netlify.app/ - 上傳 .md 檔案 → 下載 PDF

方法 2：使用 VS Code - 安裝插件：Markdown PDF - 右鍵 → Export (PDF)

流程圖 → 圖片

方法 1：使用 Mermaid Live - 網站：https://mermaid.live/ - 複製 Mermaid 代碼 → 下載 PNG/SVG

方法 2：使用截圖工具 - 在支援 Mermaid 的平台渲染（如 GitHub） - 直接截圖

HTML PPT → PDF

方法 1：瀏覽器列印 - 打開 HTML 檔案 - Ctrl+P（列印） - 選擇「另存為 PDF」 - 勾選「背景圖形」

方法 2：線上轉換 - 網站：https://www.web2pdfconvert.com/ - 上傳 HTML → 下載 PDF
