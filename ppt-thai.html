⚙️ ระบบทำงานอย่างไร? / 系統怎麼運作？

👤 ลูกค้าส่งข้อความมาที่ Telegram ส่วนตัวของคุณ

客人傳訊息到「你的個人 Telegram」

ตัวอย่าง: ลูกค้าพูดเกาหลี "오빠 오늘 갈게"

例如：客人說韓文「오빠 오늘 갈게」

📤 คุณ「กดค้าง」→「Forward」ไปที่ PPM Bot

你「長按訊息」→「轉發」到 PPM Bot

(แค่ 2 วินาที กด 2 ครั้ง)

（只需 2 秒，點兩下）

⚡ ระบบแปลอัตโนมัติ + แสดงข้อมูลลูกค้า (1วิ)

系統自動翻譯 + 顯示客戶資料（1秒）

ภาษาไทย: "พี่จะมาวันนี้" + ข้อมูล (ครั้งที่แล้วให้ 5000฿)

泰文：「哥哥今天會去」+ 客人資料（上次給 5000B）

💬 คุณพิมพ์ตอบเป็นภาษาไทยใน Web App

你在 Web App 輸入泰文回覆

ตัวอย่าง: "ออเค นะคะ" → ระบบแสดง "OK, waiting~"

例如：「ออเค นะคะ」→ 系統顯示韓文「OK, waiting~」

📋 คุณ Copy ภาษาเกาหลี → Paste กลับไปหาลูกค้า

你「複製」韓文 →「貼回」客人 Telegram

ลูกค้าเห็นแค่บัญชีส่วนตัวของคุณ ไม่รู้ว่ามีระบบช่วย

客人只看到你的個人帳號，完全不知道有系統
